# 🧠 XOR 신경망 분류기 및 시각화 도구

멀티 디맨셔널 스플릿 바이트 어레이(split-half byte array) 입력을 사용하여 클래식 **XOR(Exclusive OR) 분류 문제**를 해결하도록 설계된, 순수 NumPy 기반의 프리미엄 경량 피드포워드(feedforward) 신경망 구현체입니다.

본 프로젝트는 신경망을 **학습**(`learn`)하고, 학습된 가중치를 직렬화/역직렬화(저장 및 불러오기)하며, 견고한 파서를 통해 사용자 정의 비트 스트림을 **테스트**(`test`)하고, 우아한 학습 성능 시각화 그래프를 자동으로 생성하는 완벽한 end-to-end 명령줄 인터페이스(CLI)를 제공합니다.

---

## 🏗️ 신경망 아키텍처

이 신경망은 2차원 입력을 이진 출력에 매핑하도록 설계된 **9개의 매개변수(parameter)**를 가진 완전 연결 피드포워드(fully connected feedforward) 네트워크입니다.

*   **입력층 (Input Layer)**: 2개 입력 (분할된 바이트 배열의 비트 쌍의 두 반쪽을 나타냄)
*   **은닉층 (Hidden Layer)**: 시그모이드(Sigmoid) 활성화 함수를 사용하는 2개 뉴런 ($W_1$: $2 \times 2$ 가중치, $b_1$: $1 \times 2$ 편향)
*   **출력층 (Output Layer)**: 시그모이드(Sigmoid) 활성화 함수를 사용하는 1개 뉴런 ($W_2$: $2 \times 1$ 가중치, $b_2$: $1 \times 1$ 편향)
*   **총 학습 가능한 매개변수**: $4 + 2 + 2 + 1 = 9$개 매개변수

## 🚀 애플리케이션 실행 방법

스크립트는 명령줄 인수를 통해 **learn**과 **test**라는 두 가지 주요 모드로 실행됩니다.

### 1. 학습 모드 (`learn`)

무작위로 생성된 20개의 이진 학습 데이터 쌍을 생성하고, 신경망이 **100% 정확도**에 도달할 때까지 학습합니다(약 **284 에포크** 내외에서 완벽하게 수렴). 학습이 완료되면 가중치를 `xor_model.npz` 파일에 저장하고, 시각적으로 수려한 학습 기록 그래프를 출력합니다.

```bash
python xor_ml_visualizer.py learn
```

**주요 출력 파일:**
*   `xor_model.npz`: 학습된 가중치와 편향(`W1`, `b1`, `W2`, `b2`)이 포함된 압축 아카이브 파일.
*   `xor_training_process.png`: 손실(loss) 곡선, 정확도(accuracy) 곡선 및 9개 신경망 매개변수의 궤적(trajectory)을 보여주는 그래프.

### 2. 추론 모드 (`test`)

저장된 `xor_model.npz` 가중치를 로드하고, 길이가 동일한 두 개의 이진 입력 인수를 파싱한 뒤, 이들을 하나의 배열로 연결("붙인뒤")하고, 절반으로 분할하여 XOR 연산을 처리한 다음, 학습된 모델을 통해 이를 평가합니다.

```bash
# 단순 이진 문자열로 테스트
python xor_ml_visualizer.py test "01" "11"

# 리스트/배열 형식으로 테스트
python xor_ml_visualizer.py test "[1, 0, 1]" "[0, 0, 1]"
```

## 📊 학습 성능 및 매개변수 궤적

학습은 300 에포크 이내에 **100% 정확도**에 성공적으로 도달합니다. 학습의 동역학적 변화를 효과적으로 시각화하기 위해, 출력 그래프는 **350 에포크**로 자동 제한(trim)되어 모델의 활발한 수렴 단계와 그 이후의 안정적인 최적화 상태를 모두 한눈에 보여줍니다.

### 출력 시각화 그래프

![XOR 학습 과정 그래프](./xor_training_process.png)

*   **왼쪽 그래프 (손실 및 정확도)**: 평균 제곱 오차(MSE) 손실(빨간색)이 0으로 떨어지고 분류 정확도(파란색)가 100%로 완만하게 상승하는 과정을 보여줍니다.
*   **오른쪽 그래프 (매개변수 궤적)**: 비선형 XOR 공간을 분리하기 위해 9개의 가중치와 편향이 조정되며 최적의 값을 찾아가는 학습 여정을 시각화합니다.
